# 課題１ 当たり判定
学校の授業用  
自機の矩形の範囲を全て検査し, 敵機の範囲に被っていた場合, その敵機を指すビットを立てました.  
![demo](https://user-images.githubusercontent.com/29778890/56403455-5c089f80-629c-11e9-98fb-2975c98321e6.png)

## Usage
`gcc kadai1.c && ./a.out`  
or `./test.sh`  
* 入力データの例 (左上位置x 左上位置y 幅 高さ)  
```
自機のデータを入力: 100 100 70 100
敵機の数を入力: 3
敵機のデータを入力:
50 60 100 50
10 120 100 50
165 115 70 70
```
